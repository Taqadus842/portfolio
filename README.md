# Personal Portfolio Website

A responsive personal portfolio website showcasing my skills, projects, experience, education, certifications, and contact information. The portfolio is designed to provide an overview of my work as an AI/ML Engineer and Software Developer.

## Live Demo

You can deploy this project using Netlify or any static hosting platform.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* About Me section
* Skills showcase
* Featured projects
* Education and certifications
* Experience timeline
* Contact section
* Social media links
* Smooth scrolling and interactive navigation
* Fast loading static website

## Installation

### Clone the repository

```bash
git clone https://github.com/Taqadus842/<repository-name>.git
cd <repository-name>
```

### Run Locally

Simply open the `index.html` file in your web browser.

Or use a local development server:

Using VS Code Live Server:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## Deployment

This project is configured for deployment on Netlify.

### Deploy using Netlify

1. Push the project to GitHub.
2. Log in to Netlify.
3. Select **Add New Site** → **Import an Existing Project**.
4. Connect your GitHub repository.
5. Deploy the website.

No build command is required because this is a static website.

## Customization

You can easily customize the portfolio by:

* Updating personal information in `index.html`
* Adding or removing projects
* Replacing images inside the `assets` folder
* Modifying styles in `style.css`
* Adding new interactive features in `script.js`
